A simple C file embedding program that can embed a file into a byte array or an array of strings
and output it into a file or nicely print it in colors into the standard output.

## Table of contents
* [Quickstart](#quickstart)
* [Bugs](#bugs)
* [Make](#make)

## Quickstart
```sh
$ make
$ make install
$ fembed -h
```
